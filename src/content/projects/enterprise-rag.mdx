## System Architecture

The most critical component of this system is the data ingestion pipeline. Garbage in, garbage out.

1.  **Document Loaders:** Custom crawlers handle authentication and pull down raw HTML/PDFs.
2.  **Chunking Strategy:** Instead of fixed-size chunks, we parse the document structure (headers, paragraphs) to keep contextual boundaries intact.
3.  **Vector Store:** ChromaDB handles the billions of vector dimensions efficiently, allowing for sub-100ms similarity searches.
4.  **Generation:** The top-K retrieved documents are injected into the prompt context for the LLM to synthesize an answer and provide hard citations.
