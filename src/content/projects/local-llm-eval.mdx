## Technical Details

The core of the system is built on a simple plugin architecture. New models can be onboarded by adding a configuration file. The engine spins up a container with the required dependencies (e.g., vLLM or standard HF pipelines), runs the test suite defined in a YAML configuration, and outputs a comprehensive Markdown report.

### Key Components
- **Test Runner:** Custom pytest plugin to handle asynchronous inference calls.
- **Metric Calculators:** Wrappers around standard NLP metrics.
- **Report Generator:** Aggregates scores and builds visualizations of performance regressions.
