## Architecture Overview

The system abstracts away the complexity of the sensor network by using a gateway layer.

1. **Ingestion**: Microservices ingest high-frequency UDP packets and publish them to Kafka.
2. **Processing**: Apache Spark jobs perform windowed aggregations and anomaly detection using an Isolation Forest model.
3. **Storage**: Hot data sits in Redis for the dashboard. Cold data is dumped to Parquet in GCS via Airflow scheduled tasks.
